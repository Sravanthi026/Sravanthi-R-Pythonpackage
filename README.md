## Question Summaries

### Question 1: Descriptive Statistics R Package
**Type:** R Package Development  
**Objective:** Create a well-structured R package with 6 statistical functions with full documentation and tests.  
**Functions:** calc_mean(), calc_median(), calc_mode(), calc_q1(), calc_q3(), calc_iqr()  
**Key Features:**
- Roxygen2 documentation
- Comprehensive error handling
- Unit tests using testthat
- Standard package structure

### Question 2: SDTM DS Domain Creation
**Type:** SDTM Data Transformation  
**Objective:** Create Disposition (DS) domain using {sdtm.oak} from raw clinical trial data following CDISC standards.  
**Input:** pharmaverseraw::ds_raw  
**Output Variables:** STUDYID, DOMAIN, USUBJID, DSSEQ, DSTERM, DSDECOD, DSCAT, VISITNUM, VISIT, DSDTC, DSSTDTC, DSSTDY

### Question 3: ADaM ADSL Dataset Creation
**Type:** ADaM Data Derivation  
**Objective:** Create Subject Level Analysis Dataset with 6 derived variables using {admiral}.  
**Derived Variables:**
1. AGEGR9/AGEGR9N - Age grouping (<18, 18-50, >50)
2. TRTSDTM/TRTSTMF - Treatment start datetime
3. ITTFL - ITT population flag
4. ABNSBPFL - Abnormal baseline BP flag
5. LSTALVDT - Last known alive date
6. CARPOPFL - Cardiac adverse event flag

### Question 4: TLG - Adverse Events Reporting
**Type:** Clinical Reporting  
**Objective:** Generate publication-ready Tables, Listings, and Graphs for adverse events.  
**Deliverables:**
1. Summary Table - Treatment-emergent AEs by SOC/PT (HTML)
2. Plot 1 - AE Severity Distribution heatmap (PNG)
3. Plot 2 - Top 10 AE Incidence with 95% CI (PNG)
4. Listings - Detailed subject-level AE listing (HTML)
5. Data Export - ADAE.csv exported for Questions 5 & 6

### Question 5: Clinical Data API (FastAPI)
**Type:** Python REST API  
**Objective:** Build RESTful API for clinical trial data with dynamic filtering and risk scoring.  
**Endpoints:**
1. GET / - Welcome message
2. POST /ae-query - Dynamic AE filtering (severity, treatment arm)
3. GET /subject-risk/{subject_id} - Patient safety risk score calculation

**Features:**
- Type hints and Pydantic validation
- Risk scoring: MILD=1, MODERATE=3, SEVERE=5
- Risk categories: Low (<5), Medium (5-14), High (≥15)
- Error handling with 404 for missing subjects
- Auto-generated API docs at /docs

### Question 6: GenAI Clinical Data Assistant
**Type:** Python NLP / LLM Integration  
**Objective:** Build AI agent translating natural language queries to Pandas operations.  
**Components:**
- ClinicalTrialDataAgent class
- Schema mapping (natural language → dataset columns)
- Pattern-based query parsing
- Dynamic Pandas filtering

**Example Queries:**
- "Give me subjects with severe adverse events" → AESEV == "SEVERE"
- "Show me patients in the Placebo arm" → ACTARM == "Placebo"
- "Which subjects had moderate severity AEs?" → AESEV == "MODERATE"

## Installation & Setup

### Prerequisites
- R 4.2.0 or higher (for Questions 1-4)
- Python 3.8 or higher (for Questions 5-6)

### R Packages (Questions 1-4)

Install required packages:
```r
install.packages(c(
  "admiral", "sdtm.oak", "gt", "ggplot2", "gtsummary", 
  "tidyverse", "pharvaverse", "testthat", "roxygen2"
))
```

### Question 1: Install descriptiveStats package
```r
devtools::load_all("question_1/descriptiveStats")
# or for installation
devtools::install("question_1/descriptiveStats")

# Run tests
devtools::test("question_1/descriptiveStats")
```

### Questions 2-4: Execute R scripts

```r
# Question 2
source("question_2_sdtm/02_create_ds_domain.R")
ds_domain <- create_ds_domain()

# Question 3
source("question_3_adam/create_adsl.R")
adsl <- create_adsl()

# Question 4 (runs in order - final script exports ADAE.csv)
source("question_4_tlg/01_create_ae_summary_table.R")
source("question_4_tlg/02_create_visualizations.R")
source("question_4_tlg/03_create_listings.R")
```

### Python Setup (Questions 5-6)

**Question 5: FastAPI**
```bash
cd question_5_api

```

**Question 6: GenAI Assistant**
```bash

```
